# ML-Based Dropout Prediction and Counselling

A machine learning system that predicts student dropout risk 
and sends personalized counselling guidance via email.

## Features
- Predicts at-risk students using ML algorithms
- Automated counselling emails to students
- Early intervention to improve retention rates

## Technologies Used
- Python
- Django / Flask
- Scikit-learn / Machine Learning
- SMTP Email Integration

## How to Run
```bash
pip install -r requirements.txt
python manage.py runserver
```
